## 正式安装

### Linux平台

一般各个Linux发行版的软件库中都包含Git，直接使用相应的包管理命令安装即可，如：

Ubuntu和Debian等系统使用`sudo apt-get install git`

Fedora和Centos等系统使用`sudo yum install git`

其他Linux发行版类似，当然也可以选择源码安装，我感觉这几句是废话，毕竟Linuxer!!!

### Mac平台

一是安装`homebrew`，然后通过`homebrew`安装Git。

二是直接从`AppStore`安装`Xcode`，运行`Xcode`，选择菜单`Xcode`->`Preferences`，在弹出窗口中找到`Downloads`，选择`Command Line Tools`，点`Install`安装。

三是源码安装

### Windows平台

下载`msysgit.exe`，然后就是双击就可以自动安装啦，安装完成后，在开始菜单里找到`Git`->`Git Bash`，跳出一个命令行窗口，就说明Git安装成功！
