QuickGit

正式安装

Linux平台

一般各个Linux发行版的软件库中都包含Git，直接使用相应的包管理命令安装即可，如：

Ubuntu和Debian等系统使用sudo apt-get install git

Fedora和Centos等系统使用sudo yum install git

其他Linux发行版类似，当然也可以选择源码安装，我感觉这几句是废话，毕竟Linuxer!!!

Mac平台

一是安装homebrew，然后通过homebrew安装Git。

二是直接从AppStore安装Xcode，运行Xcode，选择菜单Xcode->Preferences，在弹出窗口中找到Downloads，选择Command Line Tools，点Install安装。

三是源码安装

Windows平台

下载msysgit.exe，然后就是双击就可以自动安装啦，安装完成后，在开始菜单里找到Git->Git Bash，跳出一个命令行窗口，就说明Git安装成功！
