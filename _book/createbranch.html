分支创建与切换

还记得版本回退里的HEAD指针吗，这个HEAD指针实际上是指向当前分支，当新创建一个分支时，实际上是新创建了一个指针，当切换到新分支时，实际上只是将HEAD指针指向了这个新分支。所以说切换速度是及其快的。

创建分支使用命令：

git branch newbranch

切换到新分支：

git checkout newbranch

注意：和撤销修改的不同，checkout后没有--

创建并切换到新分支：

git checkout -b newbranch

查看当前分支使用：

git branch

git branch命令会列出所有分支，当前分支前面会标一个*号

然后就可以在newbranch分支上正常提交，工作完成后再git checkout master切换到master分支即可，不过在newbranch提交的内容不会出现在master上，因为还没有合并分支
